# Projet_Git_Linux_Python

## Project Overview
The goal of this project is to design a professional dashboard capable of:

• Continuously retrieving and displaying real-time financial data,

• Implementing quantitative strategies and portfolio simulations,

• Clearly presenting results and key indicators in an interactive and user-friendly way.

## Objective
To design and deploy an online platform that retrieves real-time financial data, displays it in an interactive dashboard, and provides tools for backtesting and portfolio analysis.

## Division of Work
QuantA: Univariate single-asset analysis module
QuantB: Multivariate and multi-asset portfolio module
